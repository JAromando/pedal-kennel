set_success(score=1, justification=None, tool=None, group=None, report=<pedal.core.report.Report object>)[source]¶

Creates Successful feedback for the user, indicating that the entire assignment is done.

Parameters

score (number) – Arbitrary value to set the user’s score to. Defaults to 1.

compliment(message, value=0, justification=None, locations=None, tool=None, group=None, report=<pedal.core.report.Report object>)[source]¶

Create a positive feedback for the user, potentially on a specific line of code.

Parameters

  • message (str) – The message to display to the user.

  • locations (int) – The relevant line of code to reference.

  • value (float) – The number to increase the user’s score by.

  • justification (str) – A justification for why this partial credit was given.
